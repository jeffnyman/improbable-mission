# Improbable Mission

_An Homage to the 1984 "Impossible Mission" Game_

This repository will contain my work to recreate, at least in part, the ["Impossible Mission"](https://en.wikipedia.org/wiki/Impossible_Mission) game published by Epyx back in 1984.

I will basing some of this work on that done by [Krisztian Toth](http://krissz.hu/). The source code for his version is not available so I'm recreating what I can from a disassembly of one of the Atari 7800 ROM image of the game, obliquely referenced [in this thread](https://forums.atariage.com/topic/153052-impossible-mission-successful-recompile-of-source-code/). I also have, as a reference, a [Go-based version](https://github.com/acapitani/im) of Andrea Capitani's tribute to the game.

My slight rename of the game here is to make sure I run afoul of no suggestions of infringement. The use of "improbable" also reflects the likelihood of me actually being able to significantly reproduce this game in JavaScript.
